Instalation

Install nvim:

`brew install nvim`

Create config dir:

`mkdir ~/.config`

Clone this directory to `.config` dir

`cd ~/.config`

`git clone git@github.com:pedrovictorcruz/dotfiles.git .`

Install fzf and other dependencies

`brew install fd rg fzf make gcc tree-sitter exa`

Install typescritp server

`npm i -g typescript-server-config`

Enter on neovim and install packages

`:PackerInstall`
